# SmartAgent

SmartAgent 是一个面向多 Agent 协作的智能对话项目,围绕"输入理解 - 情绪分析 - 工具决策 - 回复生成 - 记忆存储"的完整链路构建。

## 项目简介

SmartAgent 的核心目标是把用户输入转化为更自然、更连贯、带有个性化风格的回复。系统保留短期对话上下文和长期记忆能力,并实现基于情绪的人格动态调整,为用户提供更智能、更贴心的对话体验。

## 核心功能

- **场景智能识别**: 自动识别对话类型(日常闲聊/任务规划/情感陪伴/知识问答),选择对应处理流程
- **多层记忆系统**: 
  - 短期记忆(Redis)保留上下文对话
  - 长期记忆(Milvus向量检索)实现历史回忆召回
  - 对话记录持久化存储(MongoDB)
- **情绪识别与人格调整**: 实时分析用户情绪状态,动态调整回复风格和语气
- **工具调用能力**: 支持网络搜索、地图查询等外部工具调用,并将结果融合到回复中
- **个性化回复生成**: 基于上下文、记忆、情绪状态生成自然流畅的个性化回复
- **智能降级策略**: Milvus服务不可用时自动降级,保证系统稳定运行

## 项目特点

- **多 Agent 协作架构**: 采用职责拆分的 Agent 设计,包括场景分类器、情绪状态分析、工具决策、回复生成等专业 Agent
- **情绪驱动的人格自适应**: 根据情绪分析结果计算人格步长,实现语气和风格的动态调整
- **阈值触发机制**: 采用 Redis 暂存 + MongoDB 持久化的双层设计,达到阈值时才更新人格配置,避免频繁写入
- **高可用设计**: 完善的降级策略,外部服务异常时系统仍可正常运行
- **模块化 Prompt 设计**: 按职责拆分的 Prompt 模板,便于维护和优化
- **灵活的工作流编排**: 支持4种工作流类型,可根据场景灵活组合 Agent 执行序列

## 技术栈

- **LLM 服务**: DeepSeek(场景分类)、DashScope(回复生成/情绪分析/向量化)
- **数据存储**: Redis(短期记忆)、MongoDB(长期记忆/用户画像)、Milvus(向量检索)
- **向量模型**: DashScope text-embedding-v3 (1024维)
