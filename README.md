# SmartAgent

SmartAgent 是一个面向多模块协作的智能对话项目，围绕“输入处理 - 状态处理 - 回复生成 - 记忆存储”的完整链路构建。

## 项目目标

项目的核心目标是把用户输入转化为更自然、更连贯、带有个性化风格的回复，同时保留短期对话上下文和长期记忆能力。

## 核心能力

- 用户输入预处理与扩写
- 短期记忆与长期记忆召回
- 人格、语气、情绪等上下文建模
- 工具结果融合
- 基于上下文生成最终回复
- 对话记录与记忆持久化

## 主要目录

- `src/main.py`：程序入口
- `src/service.py`：主服务编排，串联完整对话流程
- `src/core/`：上下文对象与回复生成核心逻辑
- `src/InputProcess/`：输入处理、扩写与记忆召回
- `src/StateToolsProcess/`：状态工具处理
- `src/Tools/`：LLM 客户端与通用工具
- `src/database/`：数据库客户端、仓储与记忆相关功能

## 典型处理流程

1. 接收用户输入
2. 创建并初始化 `Context`
3. 处理输入与召回记忆
4. 补充状态与工具信息
5. 构建最终提示词
6. 调用大模型生成回复
7. 将对话写入短期记忆与长期记忆

## 依赖的存储与服务

项目代码中使用了多种外部服务或组件，常见包括：

- Redis：短期记忆
- Milvus：向量记忆
- MongoDB：用户画像或配置数据
- 外部大模型服务：用于回复生成与向量化
