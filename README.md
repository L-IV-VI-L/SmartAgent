# SmartAgent 🤖

一个能记住你、理解你、会自己想办法的智能对话伙伴。

## 它能做什么？

**🧠 记得你说过什么**
不用反复交代背景，聊过的偏好、提过的需求，它都记着。

**💬 像朋友一样聊天**
不会一板一眼地回你，该轻松的时候轻松，该认真的时候认真。

**😊 能感知你的情绪**
你说"今天好累"，它不会给你念一段鸡汤，而是像朋友一样陪你说说话。

**🔧 需要的时候能动手**
查天气、搜信息、找地点，不用切来切去，它直接帮你搞定。

**🎯 知道什么时候该用什么招**
闲聊、查资料、陪聊、回答问题——自动判断场景，走最合适的流程。

## 技术选型

| 用途 | 技术 |
|------|------|
| 大模型服务选取 | DeepSeek · 阿里云百炼平台 |
| 短期记忆 | Redis |
| 长期记忆 | MongoDB · Milvus 向量检索 |
| API 框架 | FastAPI |

## 快速上手

```bash
# 安装依赖
pip install -r requirements.txt

# 启动 API 服务
python -m src.run_server
```

服务跑起来后，打开 `http://localhost:8000/docs` 就能看到接口文档。

## 项目结构

```
src/
├── api/          # 🌐 对外接口
├── agents/       # 🤖 各个 Agent
├── core/         # 📦 核心数据结构
├── database/     # 💾 数据库相关
├── InputProcess/ # 🔍 输入处理、记忆检索
├── prompts/      # 📝 Prompt 模板
├── Tools/        # 🔧 工具封装
└── utils/        # 🛠️ 通用工具
```
